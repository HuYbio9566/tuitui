# 360 AI 云盘八个新增工具交付报告

日期：2026-07-13

## 交付结论

8 个工具已从统一模板改造为 8 套独立产品工作区。云盘只承担入口、文件选择、权限继承和结果回存；进入工具后，信息架构、业务状态、人工复核和结果动作均依据工具特性设计。

统一体验入口：`http://127.0.0.1:8765/index.html`

## 八个工具与独立界面

| 工具 | 产品形态 | 核心演示路径 |
| --- | --- | --- |
| 文件夹智能摘要 | 文件夹分析驾驶舱 | 选择目录 -> 扫描覆盖率 -> 查看待办/风险 -> 定位来源 -> 增量扫描 |
| 多文件问答 | 证据约束对话工作台 | 选择来源 -> 提问 -> 打开页码引用 -> 处理冲突/资料不足 -> 转为简报 |
| 财务票据识别与核验 | 异常驱动财务审核台 | 选择核验任务 -> 查看异常队列 -> 确认低置信度字段 -> 阻止重复票据 -> 退回补充 |
| 合同风险审阅 | Playbook 条款与红线审查台 | 选择合同和立场 -> 定位高风险 -> 应用红线 -> 接受/忽略/转法务 -> 完成审阅 |
| 合同/文档版本对比 | 语义 Diff 工作台 | 放入多个版本 -> 浏览差异 -> 查看责任变化 -> 接受/拒绝/评论 -> 生成合并版 |
| 课件整理工具 | 课程资料编排台 | 识别教材目录 -> 课时资源分槽 -> 补答案 -> 清理重复/旧版 -> 生成课程包 |
| 教案生成工具 | 结构化教学设计编辑器 | 设置课题/材料/班情 -> 调整课堂时间轴 -> 应用局部 AI 建议 -> 教师复核 -> 导出 Word |
| 知识库一键入库 | 知识治理工作台 | 扫描候选 -> 处理重复/权限/负责人 -> 发布预检 -> 只发布合格项 -> 建立问答索引 |

## 体验一致性

统一保留 360 B 端视觉体系、工具中心导航、按钮、表单、弹窗、消息提示、云盘上下文和移动端模式。工具内部不共享业务页面 runtime，避免改名换字段式的伪差异。
