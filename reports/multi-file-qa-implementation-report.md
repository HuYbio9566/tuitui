# 多文件问答独立工作台实现报告

日期：2026-07-13

## 产品定位

多文件问答不是通用聊天框，而是围绕一组授权文件持续提问、核对来源并生成可复用产物的证据约束工作台。云盘负责提供来源和权限，工具负责来源范围、对话、引用、冲突处理与结果回存。

## 独立交互结构

- 入口：来源集合、云盘添加、问题输入、比较方案、提取事实、查找矛盾、生成纪要四类任务。
- 工作台左栏：来源解析状态、问答勾选范围、动态增删来源和权限复核提示。
- 工作台中栏：持续对话、结构化表格、冲突卡片、资料不足状态和追问输入。
- 工作台右栏：文件、页码或工作表、上下文片段、置信度和打开原文定位。
- 产物转换：把回答转为方案对比表、决策简报或项目 FAQ，并保存到云盘。

## 关键业务边界

- 每次回答记录实际使用来源，不用“已选全部资料”冒充引用。
- 用户取消某个来源后，后续回答不再使用该文件。
- 至少保留一个问答来源，避免无声进入空范围。
- 不在资料中的问题明确返回“资料不足以回答”，并建议补充来源或调整问题。
- 多份资料结论不一致时生成需人工确认的冲突状态，不擅自选边。
