# 知识库一键入库实现报告

## 产品定位

知识库一键入库是一套知识治理工作台，不是把文件上传后直接建立索引。候选资料必须完成价值、目标知识库、重复、权限、负责人和复核周期检查，才可以发布为企业可信知识。

## 独立交互模型

- 入口先扫描云盘范围，展示高价值候选和治理问题，并加载组织治理策略。
- 工作区左侧为治理队列，中间为表格型候选清单，右侧为知识详情和发布检查。
- 重复知识支持合并为新版本，保留历史访问链接和问答记录。
- 保密资料必须申请发布授权；审批中仍然阻止建立问答索引。
- 缺负责人或复核周期的资料不能发布。
- 批量发布只放行完成治理的资料，阻塞项继续留在需处理队列。

## 已实现体验

- 云盘文件夹扫描、治理策略和最近批次入口。
- 候选、可发布、需处理、待复核和已入库队列。
- 价值分、目标知识库、权限、重复、负责人和状态表格。
- AI 摘要、标签、热门问法、来源同步和可信度详情。
- 重复合并、权限申请、负责人和周期设置、批量治理设置。
- 发布前检查、合格项发布和问答索引建立。
- 桌面表格治理台与手机“队列 / 候选资料 / 治理详情”三视图。
