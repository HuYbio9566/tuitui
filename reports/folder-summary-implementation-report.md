# 文件夹智能摘要独立工作台实现报告

日期：2026-07-13

## 改动范围

- `folder-summary.html`：从通用 `app.js` 页面改为独立 runtime，加载 `folder-summary.css` 与 `folder-summary.js`。
- `folder-summary.js`：实现入口、云盘文件夹选择、预检、处理中间态、局部失败恢复、三栏证据工作台、业务动作与重新扫描对比。
- `folder-summary.css`：实现独立工作台布局、文件夹选择器、预检、扫描进度、三栏工作台、弹窗、toast 与对比抽屉样式。
- `folder-summary.test.js`：新增入口、处理中间态、三栏工作台、权限恢复、文件夹数据隔离、模板映射、增量重扫和响应式布局回归测试。

## 已实现功能

- 入口页：选择云盘文件夹、三个分析模板（领导速览、项目进展、知识归档）、最近分析。
- 云盘文件夹选择：空间树、文件夹列表、选中摘要和确认选择。
- 预检：18 文件、5 子文件夹、126 MB、格式分布、权限受限文件、OCR 图片提示。
- 处理中：目录扫描树、四个业务阶段、42% 权限受限文件局部失败、申请权限、跳过并继续、OCR 重试、取消分析。
- 结果工作台：左栏目录与覆盖，中栏领导速览、项目进展、待办事项、风险提示、关键数据、知识资产六视图，右栏来源证据。
- 业务动作：创建待办、风险已处理、保存为周报、加入知识库、分享只读报告。
- 重新扫描：增量扫描进度、变化 banner、变化对比抽屉和新旧证据展示。
