# 推推云文档与 AI 工具设计项目

这是推推云文档首页、知识工作流与云盘 AI 工具的完整设计和交互原型仓库。

## 在线体验

- GitHub Pages: https://huybio9566.github.io/tuitui/
- 当前发布入口: `index.html`
- 当前设计源文件: `designs/tuichat-cloud-docs-redesign/Cloud Docs Redesign v4 Autoplan.html`

## 项目结构

- `designs/tuichat-cloud-docs-redesign/`: 云文档重设计各版本、评审报告、预览图和完整 AI 工具子项目
- `designs/cloud-docs-knowledge-flow/`: 云文档知识流原型
- `ai-tools-demo/`: 云盘 AI 工具独立演示与测试
- `design-previews/`: 重点方案的桌面端、移动端预览
- `reports/`: 场景调研、设计交付和各工具实现报告
- `docs/`: 设计与实现计划
- `360AI云盘-8个工具演示*.zip`: 可直接交付的演示压缩包

## 主要评审资料

- `AUTOPLAN_REVIEW.md`: 当前线上版本的全维度评审摘要
- `designs/tuichat-cloud-docs-redesign/Cloud Docs Redesign v4 Autoplan Review.md`: v4 完整评审
- `designs/tuichat-cloud-docs-redesign/Yunpan AI Tools Autoplan Review 2026-07-13.md`: 云盘 AI 工具评审

本仓库中的 HTML 原型可直接在浏览器打开；根目录 `index.html` 用于 GitHub Pages 发布。
