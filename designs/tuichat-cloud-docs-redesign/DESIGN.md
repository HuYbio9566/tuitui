# 360 云文档知识工作台设计锚

## 视觉主题与氛围
企业内部知识工作台。界面应该像一张整理过的工作台面，而不是 AI 展示页：暖白底、细分隔线、低饱和蓝色只用于动作和选中态。

## DNA 注入
- 从 Notion 借用暖白纸面：`#FBFAF8`、`#F6F5F2`，用于页面底和轻分区。
- 从 Notion 借用 whisper border：`1px solid rgba(46, 42, 37, 0.10)`，替代厚重卡片边框。
- 从 Notion 借用低透明多层阴影：`0 1px 2px rgba(31,35,41,.04), 0 8px 24px rgba(31,35,41,.06)`。
- 从 Linear 借用 6px / 8px 的紧凑圆角和 `translateY(-1px)` 的 hover 位移。
- 从 Linear 借用克制动效：140-190ms ease-out，只动 `transform`、`opacity`、`background`、`border-color`。

## 排版规则
中文 UI 使用系统字体栈，字重只用 400 / 500 / 600。正文 14px 起，行高 1.5-1.75；中英文、中文与数字之间保留空格。

## 组件样式
卡片只在需要承载内容时出现，分组优先用分隔线和留白。蓝色不做大面积背景，只用在主按钮、选中态、焦点环和关键标签。

## Motion 哲学
动效服务反馈，不做表演。按钮有轻按压；hover 仅在精确指针设备上启用；Toast 与弹窗短入场，尊重 `prefers-reduced-motion`。

## 禁止项
不使用紫蓝渐变、霓虹外发光、大标题渐变字、泛泛的“AI 赋能”文案、斜体、外部中文字体或外部图片资源。
